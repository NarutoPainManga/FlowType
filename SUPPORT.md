# FlowType Support

Last updated: April 23, 2026

## What FlowType does

FlowType turns short voice notes into polished writing for email, Slack, notes, task lists, and other everyday work communication.

## Common fixes

### FlowType will not start recording

- Make sure microphone access is allowed for FlowType in iPhone Settings.
- Reopen the app and try again.
- Check the Help & Status screen inside the app.

### FlowType says setup or cloud processing is unavailable

- Confirm your internet connection is working.
- Reopen the app and refresh the status checks.
- Try again in a moment if backend services are temporarily unavailable.

### My draft did not appear in Recent

- Recent drafts are stored on this iPhone.
- If you cleared local drafts, they will no longer appear.
- If the app was interrupted before processing finished, the draft may not have been saved.

### The rewrite is wrong

- Review and edit the output before sending it.
- Shorter recordings with one clear intent usually produce the best result.
