# FlowType GitHub Pages Setup

This repo now includes a simple public docs site in the `docs/` folder for:

- Privacy Policy
- Terms of Service
- Support

## Fastest setup

1. Open the FlowType GitHub repository settings.
2. Go to `Pages`.
3. Under `Build and deployment`, choose:
   - Source: `Deploy from a branch`
   - Branch: `main`
   - Folder: `/docs`
4. Save.

GitHub will publish the site at a URL similar to:

`https://narutopainmanga.github.io/FlowType/`

## Expected page URLs

- Privacy Policy:
  `https://narutopainmanga.github.io/FlowType/privacy.html`
- Terms of Service:
  `https://narutopainmanga.github.io/FlowType/terms.html`
- Support:
  `https://narutopainmanga.github.io/FlowType/support.html`

## Important before using in App Store Connect

Replace the placeholder contact lines in the HTML pages with a real public support email or website.
